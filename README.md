# 위치기반 날씨 정보 앱

## 개발환경
- 프레임워크: flutter
- 개발언어: dart
- IDE: Android Studio
- 이미지 정보: 무료제공 이미지 파일 사용

## 기능
- 위치 정보: flutter에서 제공하는 ```geolocator``` 패키지 사용
- 날씨, 대기질 정보: [무료 API - openweathermap](https://openweathermap.org)
1. 기온
2. 날씨 상태
3. AQI(대기질 지수)
4. 미세먼지 농도
5. 초미세먼지 농도
